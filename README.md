# ポップアップ出し放題2.0

> [!NOTE]
> このREADMEは下書きです

tkinter messageboxでGUIで自分のメッセージが付いたポップアップを出しまくれるPythonソフト (おもちゃソフト)

![画面](https://raw.githubusercontent.com/ii268/Pop-updasihoudai/refs/heads/main/screenshot/main.webp)


## 使い方

1. pythonをインストール
2. ポップアップ出し放題のソースをダウンロード
3. ダブルクリック等の方法を使って実行

## ダウンロードリンク

<a href="https://raw.githubusercontent.com/ii268/Pop-updasihoudai/refs/heads/main/Pop-up2.py" download>ソースをダウンロード</a>

## 動作確認済み環境

- windows11
- Python 3.10
- tkinter 8.6 (Python同封)

## ポップアップ見た目例

デザインを選ぶことで見た目を変えることができます!

| showinfo(info) | showerror(error) |
